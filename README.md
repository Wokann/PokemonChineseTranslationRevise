# 《宝可梦》第四世代汉化修正补丁

本项目对《宝可梦》第四世代日文版进行汉化，对原汉化组使用的旧译名进行了修正，并且统一了字符码表，以便汉化版本之间连接交换及对战。

详细说明请访问 <https://xzonn.top/PokemonChineseTranslationRevise/>。
